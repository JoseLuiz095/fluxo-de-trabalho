Sensores

Sensor de Temperatura

Este é o sensor de temperatura. Ele mede a temperatura ambiente na estufa.

Sensor de Umidade do Solo

Este sensor monitora a umidade do solo para garantir que as plantas recebam a quantidade certa de água.

Sensor de Luz

O sensor de luz mede a intensidade da luz para garantir que as plantas recebam a luz adequada para o crescimento.
